About the Herb Data

This database compiles classical Ayurvedic textual data, meticulously verified from multiple authoritative sources including Charaka Samhita, Sushruta Samhita, Bhavaprakasha Nighantu, and supplemented by modern reference works such as the Ayurvedic Pharmacopoeia of India (API).

ðŸ”— Each dataset entry includes the following parameters where available:

  • Rasa (Taste)
  • Guna (Qualities)
  • Virya (Potency - Heating/Cooling)
  • Vipaka (Post-Digestive Effect)
  • Prabhava (Specific Action)
  • Dosha Karma (Effect on Vata, Pitta, Kapha)
  • Botanical Name

ðŸ§  Purpose: To create a standardized, machine-readable dataset of Ayurvedic herb properties, making this ancient knowledge more accessible for modern research, education, and digital innovation.
